6-6. 장비 입력(강화/룬/초월/소켓/인챈트 등)

장비 칸에서는 강화 수치, 룬 선택, 초월, 소켓을 빠르게 입력할 수 있습니다. 특히 강화 수치와 소켓은 마우스 좌/우클릭으로 빠른 조작이 가능합니다.

강화 수치
• 좌클릭 : +1 증가
• 우클릭 : -1 감소
소켓
• 좌클릭 : 0 → 1 → 2 → 3 (반복)
• 우클릭 : 색상 변경
  – 보라 테두리(스타프리즘) → 핑크 테두리(스타프리즘S) → 핑크 박스(온전한 스타프리즘)
특수 색상 조합 (무기 1번 소켓 / 엠블럼)
• 보라 테두리 + 노랑/초록 박스 (헬리오도르 / 그린 헬리오도르)
• 핑크 테두리 + 노랑/초록 박스 (정제된 헬리오도르 / 정제된 그린 헬리오도르)
• 노랑/초록 박스 (순수한 헬리오도르 / 순수한 그린 헬리오도르)
