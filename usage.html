사용 방법

아래 순서대로 따라 하면, 처음 설치부터 룬/캐릭/오버레이까지 한 번에 세팅할 수 있어요.

1 다운로드 exe / 룬DB(json) 받기 2 룬DB 연결 Import JSON로 룬 데이터 불러오기 3 일반 사용 예시 일일/주중/주말 체크 흐름 4 알람 결계/검은구멍/필보 타이머 5 오버레이 물물교환 / 어비스 / 레이드 6 캐릭터 세팅 직업/장신구/개조/장비/인챈트

1) 다운로드

GitHub Releases에서 exe / 룬DB(json)를 받습니다.

필수 : MobiAssistant-Portable-xxxxxx.exe
권장 : mabi_rune_db.json (기본 룬 데이터)
  • Releases의 Assets에서 exe 파일을 다운로드합니다.
  • 같은 곳에서 mabi_rune_db.json도 함께 다운로드합니다.
  • 다운로드한 exe를 실행하면 프로그램이 시작됩니다. (설치 필요 없음)

2) 룬DB 연결 (Import JSON)

처음 실행 후 룬 데이터베이스에 json을 불러옵니다.

핵심 : 메인 화면 → 룬 데이터베이스 관리 → 우측 상단 Import JSON → mabi_rune_db.json 선택
  • 메인 화면 하단의 룬 데이터베이스 관리 버튼을 클릭합니다.
  • 우측 상단의 Import JSON 버튼을 클릭합니다.
  • 파일 선택 창에서 mabi_rune_db.json을 선택 후 열기.
  • 정상 반영되면 룬 목록이 채워집니다.

* Import는 “룬 목록(데이터)”을 넣는 작업입니다. 이후 캐릭/장비 세팅에서 바로 선택 가능해집니다.

3) 일반 사용환경 예시

일일/주중/주말 퀘스트를 캐릭터별로 체크합니다.

체크 방식 : 행(퀘스트) × 열(#1~#n 캐릭터) 형태로 진행 체크
* 슬롯 수는 상단 캐릭터바의 “슬롯 추가”로 늘릴 수 있어요.
  • 일일 퀘스트 : 매일 06:00 기준
  • 주중 퀘스트 : 월 06:00 기준(주간) + 주중 컨텐츠(검은구멍/결계 등)
  • 주말 퀘스트 : 월 06:00 기준(주말에 몰아하는 것 정리용)

4) 알람

결계 / 검은구멍 / 필드보스 타이머 & 알림

알람 켜기 : 각 항목의 알람 체크박스를 ON
검은구멍 : 갱신 타이머는 - / +로 오프셋 조절 가능
  • 결계: 생성/소환/정리 구간을 기준으로 카운트가 진행됩니다.
  • 검은구멍: “갱신” 타이머를 기준으로 관리합니다.
  • 필드보스: 대기/출현 등 주기로 관리합니다.

5) 오버레이

물물교환 / 어비스 / 레이드 — 게임 위에 띄워서 체크하는 화면

오버레이 열기 : 메인 화면 하단의 물물교환 버튼, 또는 퀘스트 항목(일일물교/주간물교/어비스/레이드) 클릭

5-1) 물물교환

  • 메인 화면 하단의 물물교환 버튼을 클릭합니다.
  • 물물교환 진행 화면에서 우측 상단 물물교환 DB로 데이터베이스 화면에 들어갑니다.
  • + 새 교환식으로 교환식을 추가하고, ON 체크로 활성화하면 오버레이에 표시됩니다.
  • 오버레이 표의 #1~#n은 캐릭터별 체크이며, 완료한 캐릭터만 체크하면 됩니다.

5-2) 어비스 / 레이드

  • 주중 퀘스트의 어비스 / 레이드 항목을 클릭하면 오버레이가 열립니다.
  • 우측의 +추가로 항목을 추가하고, 캐릭터별(#1~#4 등)로 클리어 체크합니다.
  • 레이드는 여러 보스/던전 이름을 등록해두고 “오늘 누구 돌았는지” 체크용으로 쓰면 편합니다.

6) 캐릭터 세팅

슬롯 → 캐릭터 페이지에서 직업/장신구/개조/장비/인챈트를 기록합니다.

목적 : “내 캐릭터 세팅을 기록 + 룬/장비 선택을 빠르게 보기”
* 룬DB(2번)를 먼저 Import해두면, 장비/룬 선택이 훨씬 편해집니다.

6-1) 슬롯 추가 / 슬롯 선택

  • 상단 캐릭터바에서 슬롯 추가로 캐릭터 슬롯을 늘릴 수 있어요.
  • 각 슬롯을 클릭하면 해당 슬롯 기준으로 일일/주중/주말 체크가 분리됩니다.

6-2) 캐릭터 페이지 진입 / 직업 선택

  • 슬롯을 클릭하면 캐릭터 페이지에서 세팅을 기록할 수 있습니다.
  • 직업(클래스)을 선택하면 해당 클래스 기준으로 장비/룬 선택이 쉬워집니다.

6-3) 장신구 선택

  • 장신구 3종을 선택해 현재 착용 장신구를 기록합니다.
  • 장신구 선택은 “세팅 기록” + “룬 선택 참고”용입니다.

6-4) 개조 / 스킬 태그 / 펫

  • 개조 번호를 눌러 현재 선택/강조 상태를 기록합니다.
  • 스킬 태그는 내 세팅 방향(연타/강타/방해 등)을 표시하는 용도입니다.
  • 펫 영역에는 메모처럼 효과/태그를 적어둘 수 있어요.

6-5) 클래스 전용 장비 (세분화 / 간소화)

  • 기본 화면은 간단하게 “무기/엠블럼/목걸이/반지/반지”를 선택합니다.
  • 우측 상단 세분화를 누르면 인챈트/소켓/초월 등 상세 입력이 펼쳐집니다.
  • 다시 간소화로 접어서 깔끔하게 볼 수 있어요.
팁 : “세분화는 디테일 기록용”, “간소화는 빠르게 보기용”으로 쓰면 가장 편합니다.

※ 이 문서는 현재 UI 스크린샷 기준으로 작성되었습니다. 버튼 이름/배치가 업데이트되면 스크린샷을 교체해주면 됩니다.
