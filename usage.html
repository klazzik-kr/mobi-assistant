사용 방법

스크린샷을 추가하기 전까지는 “틀리지 않는 범위”로 안내만 둡니다.

← 다운로드 페이지로

1) 다운로드

  1. Releases (latest)로 이동합니다.
  2. Assets에서 MobiAssistantPortable.exe를 다운로드합니다.
  3. 필요한 경우 rune.json도 함께 다운로드합니다.
[추가 예정] Releases 화면 스크린샷: images/usage-01.png

2) 실행

  1. MobiAssistantPortable.exe를 실행합니다.
  2. Windows SmartScreen 경고가 뜰 수 있습니다(개인 제작/서명 없음).
[추가 예정] 프로그램 실행 화면 스크린샷: images/usage-02.png

3) 룬 데이터 적용

rune.json 적용 방식은 앱 UI 기준으로 안내가 가장 정확합니다. 스크린샷을 추가한 뒤, 이 섹션을 “버튼 위치/메뉴 이름” 기준으로 확정합니다.

[추가 예정] 룬 불러오기/가져오기 화면 스크린샷: images/usage-03.png
  • OneDrive 동기화 폴더에서는 파일 잠김/권한 문제로 오동작할 수 있어 일반 폴더 사용을 권장합니다.
  • rune.json이 보이지 않거나 적용이 안 되면 파일명/형식을 먼저 확인하세요.
