usage.html

이 페이지는 포터블 exe 실행 + 룬DB(JSON) 가져오기 + 일일/주간 체크 + 오버레이 사용 흐름을 스크린샷 기반으로 정리한 문서입니다.

← index로 돌아가기

1) 다운로드

  • Releases 페이지의 Assets에서 파일을 받습니다.
  • 기본적으로 exe(프로그램)와 mabi_rune_db.json(룬DB) 두 개가 있습니다.
팁
exe만 받아도 실행은 되지만, 룬DB를 같이 쓰려면 mabi_rune_db.json을 가져오면 편합니다.

2) 룬DB(JSON) 연결

2-1. 메인에서 룬 데이터베이스로 이동

2-2. JSON 파일 선택

참고
룬DB는 Import/Export로 관리합니다. (친구에게 배포할 때도 JSON만 공유하면 동일하게 세팅 가능)

3) 기본 사용환경 예시

  • 메인 화면에서 캐릭터 슬롯, 일일/주간/주말 체크, 상점 체크를 한 화면에서 관리합니다.
  • 슬롯 수(#1~#6)는 캐릭터바에서 늘릴 수 있습니다.

4) 알람

  • 결계 / 검은구멍 / 필드보스 알람을 켜고, 남은 시간을 확인할 수 있습니다.
  • 검은구멍은 갱신 타이머 기준으로 관리됩니다. (예: 남은 시간 표시)
팁
알람은 “켜두고 보는 용도”에 가깝고, 체크박스(퀘스트/오버레이)는 “완료 여부 기록” 용도라고 보면 됨.

5) 오버레이

오버레이는 메인 체크보드에서 특정 항목(일일물교/주간물교/어비스/레이드 등)을 눌렀을 때 뜨는 “상세 체크창”입니다.

5-1. 물물교환 오버레이

정리
물물교환은 DB에 교환식을 등록 → ON 활성화 → 오버레이에서 캐릭터별 체크로 관리하는 흐름입니다.

5-2. 어비스/레이드 오버레이

팁
오버레이 항목은 “직접 쓰는 리스트”라서, 본인이 도는 어비스/레이드 이름만 추가해두면 됨.

추가: 캐릭터 세팅(직업/장신구/개조/장비)

6-1. 슬롯 추가 / 슬롯 확장

6-2. 캐릭터 페이지에서 직업 선택

중요
직업을 선택하면 메인페이지 캐릭터 슬롯/표시가 함께 바뀝니다.

6-3. 장신구 선택 → 장비칸에 자동 반영

6-4. 스킬 개조 선택

6-5. 스킬 태그 / 펫 메모

6-6. 장비 입력(강화/룬/초월/소켓/인챈트 등)

주의
이 페이지의 스크린샷 파일명은 저장소의 /usage_image 폴더에 있는 파일명을 그대로 사용합니다.
파일명을 바꾸면, 여기 img 경로도 같이 바꿔야 합니다.
