다운로드

⬇ 최신 버전 다운로드 (GitHub Releases)
Releases 페이지의 Assets에서 MobiAssistantPortable.exe 와 rune.json을 받으면 됩니다.

빠른 사용법

  1. MobiAssistantPortable.exe 다운로드
  2. rune.json을 exe와 같은 폴더에 둠
  3. exe 실행
Windows SmartScreen 경고가 뜰 수 있습니다(개인 제작/서명 없음).

업데이트

  • 새 버전이 나오면 Releases만 갱신하면 됩니다.
  • 페이지 링크는 그대로 유지됩니다.
